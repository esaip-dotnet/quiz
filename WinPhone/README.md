L'application WindowsPhone permet de récupérer et écrire des informations dans une API.


# Modification du code selon vos besoins 
Ce code permet de lire et écrire un fichier JSON présent dans une api, il est adapable en fonction des besoins. 


## Modifier les urls redirigeant vers l'API 
Dans le code 2 urls sont présentes une dans la méthode GetData() et l'autre dans Valider_click().
Remplacez les par votre url.

##Modifier la classe "WeatherObject" en fonction du contenu de votre fichier json 
Un site peut vous aidez pour cela : http://json2csharp.com/
N'oubliez pas de modifier également l'objet JSON dans la méthode GetData()

#Quiz

Bienvenue sur votre application Quiz !
Ce quiz interactif vous permettera de tester vos connaissances en vous amusant !

Avec les nouvelles technologies nous pouvons rendre votre experience encore plus enrichissante !


#Comment Jouer ? 

Pas dur ! il vous suffit de selectionner le mode de jeu que vous souhaitez utiliser !

-[Kinect]
-[WPhone]
-[LeapMotion]

Vous n'avez qu'a lancer l'application et puis tout est indiqué !

#Comment lancer l'application ?

Il vous suffit de cliquer les Quiz.exe fournit avec le Programme !


#Installation

Pour installer ce Quiz, il vous suffit comme indiqué plus haut de double cliquer sur les executables Quiz 


#Quelle Plateforme ?

Vous preferez jouer avec vos doigts ? Allez sur WindowsPhone !
Vous aimez manipuler les choses avec votre mains ? Allez sur LeapMotion ! 
Vous voulez bouger et répondre avec votre corps ? Allez sur Kinect !
